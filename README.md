# 🎮 游戏存档备份工具

一个功能强大的图形界面应用程序，专为游戏玩家设计，提供智能的存档备份和恢复功能。支持手动/自动备份区分、热键操作和配置持久化。

## ✨ 功能特性

### 🔧 核心功能
- 🎨 **现代化界面** - 使用PyQt6构建的美观图形界面
- ⏰ **智能自动备份** - 可配置时间间隔的定时备份
- ⌨️ **全局热键支持** - F2手动备份，F3智能恢复
- 📝 **备份类型区分** - 手动备份和自动备份独立管理
- 🔄 **智能恢复优先级** - F3优先恢复手动备份，无手动备份时恢复自动备份

### 🛠️ 实用功能
- 💾 **配置持久化** - 所有设置自动保存到JSON文件
- 📁 **多目录支持** - 灵活选择游戏存档和备份目录
- 📋 **备份历史管理** - 查看最近10个备份记录
- 🎯 **一键恢复** - 快速恢复到最新备份版本
- 🔔 **实时状态反馈** - 状态栏和声音提示

## 🎯 智能备份系统

### 备份类型

| 类型 | 触发方式 | 命名格式 | 优先级 |
|------|----------|----------|--------|
| 手动备份 | F2键或点击按钮 | `manual_backup_YYYY-MM-DD_HH-MM-SS` | ⭐⭐⭐ |
| 自动备份 | 定时执行 | `auto_backup_YYYY-MM-DD_HH-MM-SS` | ⭐⭐ |

### 恢复优先级逻辑

```
F3恢复请求
    ↓
检查最新手动备份
    ├── ✅ 存在 → 恢复最新手动备份
    └── ❌ 不存在 → 恢复最新自动备份
```

## 🚀 快速开始

### 系统要求
- **操作系统**: Windows 10/11
- **Python版本**: Python 3.13+
- **权限要求**: 管理员权限（用于全局热键注册）

### 安装步骤

#### 1. 安装依赖
```bash
# 创建虚拟环境
uv venv .venv

# 安装项目依赖
uv pip install -e .
```

#### 2. 启动程序
```bash
# 激活虚拟环境 (Windows)
.venv\Scripts\activate

# 运行程序
python main.py
```

#### 3. 快速配置
1. 点击"浏览"选择游戏存档目录
2. 选择备份保存目录
3. 设置自动备份间隔（建议15-30分钟）
4. 启用自动备份功能
5. 点击"保存设置"

#### 4. 开始使用
- **F2键**: 立即创建手动备份
- **F3键**: 智能恢复到最新备份
- **自动**: 按设定间隔自动备份

## ⚙️ 配置系统

### 配置文件位置
程序启动时会自动创建 `config.json` 文件

### 配置格式
```json
{
  "current_game": "游戏名称",
  "games": {
    "游戏名称": {
      "game_name": "游戏名称",
      "source_directory": "C:/存档路径",
      "backup_directory": "D:/备份路径",
      "backup_interval_minutes": 30,
      "auto_backup_enabled": true,
      "last_backup_time": "2024-05-16 14:30:00",
      "created_date": "2024-05-16"
    }
  },
  "last_updated": "2024-05-16 14:30:00"
}
```

### 配置说明
- **source_directory**: 游戏存档源目录
- **backup_directory**: 备份文件保存目录
- **backup_interval_minutes**: 自动备份间隔（分钟）
- **auto_backup_enabled**: 自动备份启用状态
- **last_backup_time**: 最后备份时间戳

## 📝 备份命名规则

### 手动备份
- **格式**: `manual_backup_YYYY-MM-DD_HH-MM-SS`
- **示例**: `manual_backup_2024-05-16_14-30-00`
- **触发**: F2热键或点击备份按钮
- **用途**: 重要进度前的保护性备份

### 自动备份
- **格式**: `auto_backup_YYYY-MM-DD_HH-MM-SS`
- **示例**: `auto_backup_2024-05-16_14-30-00`
- **触发**: 按设定时间间隔自动执行
- **用途**: 定期进度保存

### 恢复策略
- **F3恢复**: 优先使用最新手动备份
- **回退机制**: 无手动备份时使用最新自动备份
- **安全保障**: 双重保护，永不丢失进度

## 🔧 故障排除

### 热键问题
**问题**: F2/F3热键不起作用
**解决方案**:
1. 以管理员身份运行程序
2. 检查是否有其他程序占用热键
3. 暂时禁用安全软件测试

### 备份失败
**问题**: 备份创建失败
**解决方案**:
1. 检查源目录是否存在和可读
2. 确认备份目录有写入权限
3. 检查磁盘空间是否充足
4. 查看防病毒软件设置

### 恢复问题
**问题**: 无法恢复到期望的备份
**解决方案**:
1. 在备份历史中确认备份存在
2. 检查源目录是否被其他程序占用
3. 手动从备份目录复制文件

### 配置问题
**问题**: 设置无法保存
**解决方案**:
1. 检查程序目录写入权限
2. 手动编辑config.json文件
3. 重新安装程序
